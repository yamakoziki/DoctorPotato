# じゃがいもマスター 🥔
**ネパール農家向け ジャガイモ栽培技術 評価ツール**

ネパール中山地の小規模農家が、ジャガイモ栽培に関する知識と技術を自己評価できるウェブアプリです。10カテゴリ・30問のクイズ形式で技術レベルを診断し、弱点に応じた改善アドバイスを提供します。

---

## 起動方法

ビルド不要。`nepal-assessment-v2.html` をブラウザで開くだけで動作します。サーバー・依存パッケージは一切不要です。

---

## 使い方

### 1. スタート画面
- **名前・地区・メールアドレス**（任意）を入力
- 評価したいカテゴリをカードから選択（1つ以上必須）
- 「評価を開始する」ボタンでクイズ開始

### 2. クイズ画面
- 選択したカテゴリから問題がランダムに出題（カテゴリあたり3問）
- A〜D の4択から1つを選択
- 回答後に即座に正誤フィードバックと解説が表示される
- 「次の問題へ」で進む（最終問題後は結果画面へ）
- 左上の「← もどる」ボタンで最初からやり直せる

### 3. 結果画面
- 総合スコア（0〜100点）と評価レベルが表示される
- カテゴリ別スコアの一覧表とバーチャートで弱点を可視化
- スコアに応じたコメントと改善アドバイスが自動生成される
- メールアドレスを入力していた場合、結果レポートをメール送信できる

## 評価カテゴリ（10分野）

| カテゴリ | 内容 |
|---|---|
| 🌱 土壌管理 | pH・深耕・堆肥施用 |
| 🌿 種芋準備 | 催芽・コルク化・認定種芋 |
| 🪣 植え付け | 時期・深さ・株間 |
| 💧 施肥・灌水 | カリウム・窒素バランス・灌水タイミング |
| 🐛 病害虫対策 | 疫病・アブラムシ・輪作 |
| ⛏️ 培土・除草 | 土寄せ時期・緑化防止・マルチング |
| 🥔 収穫・判断 | 収穫適期・損傷防止・ソラニン |
| 🏚️ 貯蔵・出荷 | 温度管理・キュアリング・等級分け |
| 💰 販売・市場 | 価格情報・共同出荷・出荷時期 |
| 📚 技術学習 | 普及員活用・試験区・栽培記録 |

---

## 評価レベル（5段階）

| スコア | レベル |
|---|---|
| 90〜100点 | 🌟 Outstanding（卓越） |
| 80〜89点 | 🎖 Excellent（優秀） |
| 70〜79点 | 🌿 Good（良好） |
| 60〜69点 | 🌱 Needs Improvement（要向上） |
| 60点未満 | 🌰 Unsatisfactory（入門段階） |

---

## 問題の難易度

各問題には難易度が設定されています。

- ★☆☆ 基礎（lv 1）
- ★★☆ 応用（lv 2）
- ★★★ 上級（lv 3）

---

## メールレポート機能

結果画面からスコアとカテゴリ別成績をまとめたレポートを `mailto:` リンクで送信できます。メールアプリが起動し、件名・本文が自動入力されます（実際の送信はメールアプリ側で行います）。

---

## 技術仕様

- **フロントエンド:** HTML5 / CSS3 / Vanilla JavaScript（依存ライブラリなし）
- **フォント:** Google Fonts（Zen Antique Soft・Noto Sans JP）
- **対応環境:** ES6+対応モダンブラウザ、モバイルファースト設計
- **構成:** 単一HTMLファイル（CSS・JSすべて埋め込み）
